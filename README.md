## アプリ概要

ソフトウェアテストをUI上で作成管理するためのアプリ

### 作成した背景

ソフトウェアテストを作成する際にコードをVSCodeを用いて管理していたがより簡単に管理しやすくするためにデスクトップアプリを作成した。
